# Tableau 시각화 툴
기본적으로 Groupby가 적용된다고 볼 수 있다

## 용어
- 집계(agg) : 테이블에 올리는 자동집계된 값 데이터
- 차원 및 측정값 : 집계를 적용할 수 있으면 측정값/문자나 날짜 등은 차원
    - 불연속형 차원 : str
    - 연속형 차원 : date
    - 불연속 측정값 : int이긴 한데..잘 안쓴다
    - 연속 측정값 : int로 범위에 포함되어있는 숫자값들
- 계층
