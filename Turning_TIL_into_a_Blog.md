# TIL 블로그화
github 레포지토리를 블로그화 하는걸 지인을 통해 알게 되었습니다. 여러 장점이 있는 것으로 보여서 진행해보고자 하는데 우선 설치해야할 것이 많다는 것을 알았습니다.

# install
1. Node.js 설치
Node.js에서 LTS 버전을 다운로드 후 설치(npm도 자동으로 함께 설치됩니다)

    - 설치 확인
    ```
    node -v
    npm -v
    ```

    - 버전 권장 사항
        - Node.js : LTS(16.x 이상 권장)
        - npm : 8.x 이상 권장

2. lerna, yarn 설치
- sudo npm install -g lerna yarn

3. 환경설정 확인
(미진행)
